iOS Guide - Exam Portal

Method 1: Using "Web Server" App (Recommended)

  1. Install "Web Server" from App Store (it's free)
  2. Open the app
  3. Click the "+" button to add your exam folder
  4. Select the folder containing your exam files
  5. Click "Start Server"
  6. Open Safari and go to: http://localhost:8080

Method 2: Using "Files" App

  1. Open the "Files" app on your iPhone
  2. Navigate to where you saved the exam files
  3. Tap on "index.html"
  4. It will open in Safari

Important Notes:

  • Keep all files in the same folder
  • Don't delete any files
  • The exam works completely offline
  • You can close and reopen anytime

If you have issues:

  1. Make sure all files are in the same folder
  2. Try using Method 1 (Web Server app) if Method 2 doesn't work
  3. If still having problems, contact your teacher
